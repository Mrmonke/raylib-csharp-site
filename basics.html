Getting Started with Raylib C#

This section covers the basics: how to set up Raylib with C#, write your first program, and understand the game loop.

Installing Raylib on Windows

To install Raylib for C# on Windows, use the following command:

dotnet add package Raylib-cs

Make sure you have the .NET SDK installed. You can download it from here.

Installing Raylib on macOS

To install Raylib for C# on macOS, first install Homebrew and then use:

brew install raylib

Then add the Raylib package to your .NET project:

dotnet add package Raylib-cs

Installing Raylib on Linux

On Linux, install Raylib via your package manager:

# Ubuntu/Debian
sudo apt install libraylib-dev

# Arch Linux
sudo pacman -S raylib

Then add the Raylib package to your .NET project:

dotnet add package Raylib-cs
